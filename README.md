## Getting Started

First, [install pnpm](https://pnpm.io/installation)

```bash
npm install -g @pnpm/exe
```

and

```bash
pnpm install
```

then run the development server:
```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
